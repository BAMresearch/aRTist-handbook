aRTist File Format
------------------

An aRTist file is used to save the complete program status so that the working state can be restored without further dependencies using any compatible |artist| installation.

aRTist files using the file name extension ``.artist``.

The aRTist file is a zipped folder with a scene description and its dependent files.
